# 🚀 AI Resume Analyzer

## 📌 About Project
This project analyzes resumes and gives ATS score, skill extraction, and suggestions.

## 💡 Features
- Upload Resume (PDF)
- Extract Skills
- ATS Score Calculation
- Suggestions for Improvement

## 🛠 Tech Stack
- Python (Flask)
- React
- NLP (spaCy)

## ▶️ How to Run

### Backend
cd backend  
pip install -r requirements.txt  
python app.py  

### Frontend
cd frontend  
npm install  
npm start
